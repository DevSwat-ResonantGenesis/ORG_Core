## 🔒 Required Environment Variables for Production

Add these to your .env.production:

```bash
# Frontend URL (REQUIRED)
AUTH_FRONTEND_URL=https://yourdomain.com

# Alternative names (some services use these)
FRONTEND_URL=https://yourdomain.com
BILLING_FRONTEND_URL=https://yourdomain.com

# CORS Origins
NODE_ALLOWED_ORIGINS=https://yourdomain.com,https://app.yourdomain.com

# Email sender
AUTH_EMAIL_FROM_ADDRESS=noreply@yourdomain.com
```
